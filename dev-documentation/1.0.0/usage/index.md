# Usage Examples

This section provides end-to-end examples of how to use CEON to represent real-world data in the three use case domains of the Onto-DESIDE project.

Each example includes:

- A description of the scenario
- Sample data in Turtle format
- SPARQL queries to retrieve information from the data

## Use Cases

- [Construction](construction.md) — material passports and building component lifecycles
- [Electronics](electronics.md) — product traceability and end-of-life processing
- [Textile](textile.md) — fibre composition and recycling value chains
